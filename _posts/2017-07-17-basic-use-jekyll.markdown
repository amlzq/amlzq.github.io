* 
```
安装了 Jekyll Compose 后，Jekyll 会额外提供一些命令，便于发布管理博文。

创建草稿：

$ jekyll draft
创建新博客：

$ jekyll post
创建新博客：

$ jekyll page
发布草稿：

$ jekyll publish
撤销发布：

$ jekyll unpublish
```
